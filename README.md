# Enterprise CRM System

מערכת CRM מתקדמת לניהול עסקים עם יכולות ניהול אנשי קשר, פגישות, משימות ועוד.

## מבנה הפרויקט

```
crmhtml/
├── index.html          # נקודת הכניסה הראשית לאפליקציה
├── css/
│   └── styles.css      # כל עיצוב האפליקציה
├── js/
│   └── app.js          # כל הלוגיקה של האפליקציה
├── assets/
│   └── images/         # תמונות ומשאבים גרפיים
├── crm.html            # קובץ מקורי (deprecated - לא בשימוש)
└── README.md           # קובץ זה

```

## תיקיות

- **css/** - קבצי עיצוב (CSS)
- **js/** - קבצי JavaScript
- **assets/** - משאבים סטטיים (תמונות, פונטים וכו')
  - **images/** - קבצי תמונות

## איך להתחיל

1. פתח את הקובץ `index.html` בדפדפן
2. המערכת עובדת מול localStorage של הדפדפן
3. כל הנתונים נשמרים מקומית במכשיר שלך

## תכונות

- ניהול אנשי קשר
- ניהול פגישות ולו"ז
- ניהול משימות
- מערכת מסמכים
- לוח שנה אינטראקטיבי
- ניהול צוות
- דוחות וניתוח נתונים
- הגדרות מערכת

## טכנולוגיות

- HTML5
- CSS3 (עם CSS Variables)
- Vanilla JavaScript (ללא תלויות חיצוניות)
- Font Awesome 7.0.1 לאייקונים
- LocalStorage לשמירת נתונים
